# c11lang
Originally created as C++11 feature demo, but extended to C++17.

# Build instructions
Compile and run with g++ on Linux.
Make file included.
Code samples in this demo:

* any type
* auto types
* Functors
* Lambdas
* Tuples
* Variadic expansions
* Variadic templates
* Variadic types
* Variadic arguments access
